第二部分 生活原则

1 拥抱现实，应对现实

世界上最重要的事情是理解现实如何运行，以及如何应对现实。面对这一过程的心态至关重要。我发现很有帮助的做法是，把生活想象为一场游戏，我面临的每个问题都是一个需要破解的谜。

1.1 做一个超级现实的人

我逐渐明白，从生活中获取尽可能多的东西，并不只是一个要为此而工作的更辛苦的问题。这更多的是一个有效工作的问题，因为有效工作能把我的能力提升几百倍。

  • 梦想+现实+决心=成功的生活

1.2 真相（或者更精确地说，对现实的准确理解）是任何良好结果的根本依据

当真相与愿望不符时，大多数人抗拒真相。这很糟糕，因为好东西会自己照顾自己，而理解和应对不好的东西才是更重要的。

1.3 做到头脑极度开放、极度透明

  • 对于快速学习和有效改变而言，头脑极度开放、极度透明是价值无限的
    • 避免误解
    • 提高反馈循环的效率
  • 不要担心其他人的看法，使之成为你的障碍
    • 就像公开发言，一开始会难堪，但练的越多越能应付自如
  • 拥抱极度求真和极度透明将带来更有意义的工作和更有意义的人际关系
    • 只要多练习，绝大多数人会感觉这样做很有收获

1.4 观察自然，学习现实规律

自然比我更聪明，我努力让自然教我认识现实规律

  • 大脑的进化让我们能够思考现实的运行方式（超越自身，从更高维度看待现实，并加以总结）
  • 理解任何东西的两种视角
    • 自上而下
    • 自下而上
  • 不要固守你对事物『应该』是什么样的看法，这将使你无法了解真实的情况
    • 什么是好什么是坏
    • 我的人生目标是什么
    • 当面对最重要的抉择时我该怎么做
  • 一个东西要『好』，就必须符合现实的规律，并在促进整体的进化，这能带来最大的回报
  • 进化是宇宙中最强大的力量，是唯一永恒的东西，是一切的驱动力
    • 完美是不存在的，它是一个目标，激发永不停息的进化过程
    • 基因、文化、科技、语言的进化
  • 不进化就死亡

1.5 进化是生命最大的成就和最大的回报

个体的人既是一切又什么都不是，这是一个巨大的悖论。

  • 进化创造了
    • 促使个体追求自身利益、最终使整体进步的激励和互动机制
    • 自然选择过程
    • 快速的试验和适应
  • 个体的激励机制必须符合群体的目标
  • 现实为了整体趋向最优化，而不是为了个体
  • 通过快速试错以适应现实是无价的
    • 三种学习方式：以记忆为基础的（有意识的信息积累）、潜意识的（无意识的经验学习）、与人类思考无关的（基因本身的进化）
  • 意识到你既是一切又什么都不是，并决定你想成为什么样子
    • 我们的生命终止了，对于我们而言，全世界也就消失了
    • 人类在空间和时间尺度上的渺小
  • 你的未来取决于你的视角

1.6 理解自然提供的现实教训

追逐诱饵要求我们必须进化，而且对所有人来说重要的是进化，而不是回报本身。

  • 把你的进化最大化
  • 记住『没有痛苦就没有收获』
  • 自然的一项根本法则是，为了赢得力量，人必须努力突破极限，而这是痛苦的

1.7 痛苦+反思=进步

你面临的挑战将考验你，强化你。如果你没有经历失败，就说明你没有努力突破极限，而如果你不努力突破极限，你就不能最大限度地挖掘自己的潜能。如果你选择在这条常常很痛苦的个人进化之路上拼搏，你将自然而然地不断『更上一层楼』。

  • 迎接而非躲避痛苦，从痛苦中学习，将让人『步入新境界』
    • 找到、接受，并学会如何应对你的弱点；
    • 更喜欢周围的人对你坦诚，而不是隐瞒对你的负面看法；
    • 展现真实的自我，而不是强行把弱点伪装成优势
  • 接受严厉的爱

1.8 考虑后续与在后续的结果

如果人们过度重视决策的直接结果而忽视后续和再后续的结果，他们就很难实现目标

1.9 接受结果

不管在生活中遇到什么情况，如果你能负起责任，进行良好的决策，而不是抱怨你无法控制的东西，你将更有可能成功并找到幸福

  • 可控和不可控的因素（内控点）

1.10 从更高的层次俯视机器

目标->机器（设计和人）->结果->目标

  • 把自己想象成一部在大机器里运转的小机器，并明白你有能力改变你的机器以实现更好的结果
  • 通过比较你实现的结果和目标，你就能确定如何改进你的机器
  • 区别作为机器设计者的你和作为机器工作者的你
  • 大多数人犯下的最大错误是不客观看到自己和其他人，这导致他们一次次地栽在自己或其他人的弱点上
  • 成功的人能超越自身，客观地看待事物，并管理事物以塑造改变
    • 面对自身缺点的四种选择
      • 你可以否认缺点（这是大多数人的做法）
      • 你可以承认缺点并应对缺点，努力把缺点转化为优势（能不能成功取决于你的自我改变能力）
      • 你可以承认缺点并找到绕过缺点的方法
      • 或者，你可以改变你的目标
  • 在不擅长的领域请教擅长的其他人，这是一个你无论如何都应培养的出色技能，这将帮助你建立起安全护栏，避免自己做错事
  • 因为客观看待自身很困难，所以你需要依赖其他人的意见，以及全部证据
  • 如果你的头脑足够开放，足够有决心，你几乎可以实现任何愿望
不好 好
拒绝直面『严酷的现实』 直面『严酷的现实』
担心自己的形象 关心如何实现目标
只根据直接结果做决策 根据直接结果、后续结果和再后续结果做决策
让痛苦阻碍自己进步 理解如何管理痛苦以取得进步
不自己负起责任，也不会从其他人那里找原因 从自己和其他人那里找原因，负起责任

2 用五步流程实现你的人生愿望

3 做到头脑极度开放

3.1 认识你的两大障碍

  • 影响合理决策的两大障碍
    • 自我意识
    • 思维盲点
  • 自我意识的解释
    • 大脑原始区域（杏仁核、颞叶）渴望赞誉，抵制批评
    • 大脑高级区域（前额皮层）进行逻辑和推理，可以『理解』建设性的批评
    • 『有逻辑和有意识的你』 vs 『情绪化和潜意识的你』，切莫让『想要自己正确』的需求压倒『找出真相』的需求
  • 思维盲点的解释
    • 受限于个人的思维方式、认知和理解事物的能力
    • 例子：线性思维或发散思维；大图景或小细节
  • 适应和进化的三种途径
    • 训练自己的头脑以反直觉的方式思考（例如有创造力的人通过自律和练习变得更有条理）
    • 利用辅助机制（例如程序化的提醒器）
    • 在自己的短板上，依靠擅长者的帮助

3.2 奉行头脑极度开放

  • 从『我是对的』到『我怎么知道我是对的』
    • 你在决策时看到的情况也许并不是最符合事实的情况
    • 有效的探析各种不同的观点和不同的可能性，而不是让自我意识或思维盲点阻碍你
  • 如何做到头脑极度开放
    • 诚恳的相信你也许并不知道最好的解决办法是什么，并认识到，与你知道的东西相比，能不能妥善处理『不知道』才是更重要的
    • 认识到决策应当分成两步：先分析所有相关信息，然后决定
    • 不要担心自己的形象，只关心如何实现目标
    • 认识到你不能『只产出不吸纳』
    • 认识到为了能够从他人的角度看待事物，你必须暂时悬置判断，只有设身处地，你才能合理评估另一种观点的价值
    • 谨记，你是在寻找最好的答案，而不是你自己能得出的最好答案
    • 搞清楚你是在争论还是在试图理解一个问题，并根据你和对方的可信度，想想哪种做法最合理

3.3 领会并感激：深思熟虑的意见分歧

  • 不是在争论，而是在开放地探求事实
  • 通过分歧来进行学习，保持头脑开放但也要坚定
    • 复述对方的问题
    • 两分钟法则（两分钟内不允许打断对方）
  • 值不值得解决分析
    • 可信的人应当花时间来探讨观点
    • 通过互相理解，保持分析，来缓解僵局
  • 为什么不愿表达分歧
    • 『较低层次的你』误以为分歧是冲突

3.4 和可信的、愿意表达分歧的人一起审视你的观点

  • 审视多个可信者的观点，专家也会犯错误
  • 为最坏的情况做准备，以尽量使其不那么糟糕

3.5 识别你应当注意的头脑封闭和头脑开放的不同迹象

头脑封闭的人 头脑开放的人
不喜欢自己的观点被挑战 更想了解为什么会出现分歧
更喜欢做『低自信程度』的陈述而不是提问 真诚地相信自己可能是错的，提出真诚的问题
更关心是否被理解，而不是理解其他人 有必要从对方的角度去看事物
『我可能错了……但这是我的观点』 知道何时做陈述，何时提问
阻挠其他人发言 更喜欢倾听而不是发言，鼓励其他人发表观点
难以同时持有两种想法 同时思考多个相互冲突的概念，反复权衡其相对价值
缺乏深刻的谦逊意识 看待事物时，时刻担忧自己可能是错的

3.6 理解你如何做到头脑极度开放

  • 如何练习头脑开放
    • 经常利用痛苦来引导自己进行高质量的思考
    • 将头脑开放作为一种习惯
    • 认识自己的思维盲点
    • 假如很多可信的人都说你正在做错事，只有你不这么看，你就要想想自己是不是看偏了

4 理解人与人大不相同

因为不同人的大脑构造不同，所以我们体验现实的方式千差万别，而任何人的单一体验本质上都是存在扭曲的。

我为什么开始研究神经科学

问题在于，理念型的人大略的设想出要做什么，并希望细节型的人自己找到办法，当后者找不到办法时，前者就觉得后者毫无想象力，而后者觉得前者自己都没搞清楚。更糟糕的是，双方都不自知，都觉得对方和自己的思维方式是一样的。简言之，两方相争，我们陷入僵局，而所有人都觉得是别人的错，觉得对方盲目、顽固，或者只是愚蠢。

  • Ray Dalio需要的聪明人
    • 独立的思考者
    • 有创造力，概念性强，常识性强
  • 不同类型的人与『沟通问题』

4.1 明白你与其他人的思维方式能带来的力量

我们的分歧不是沟通不良造成的，相反，我们不同的思维方式导致了沟通不良。

  • 我们拥有各种天生特征，既可能帮助自己也可能伤害自己，取决于如何运用
    • 棒球卡
    • 理解大脑这部『思维机器』
      • 人脑的复杂程度超出我们的想象
      • 新生儿的大脑里就包含了数亿年生物进化累积的成果
      • 所有哺乳动物、鱼类、鸟类、两栖动物、爬行动物的脑都拥有同一套伟大的构造
      • 这一『所有脊椎动物共有的脑』的进化是自下而上进行的，也就是说，较低的区域是进化意义上最古老的，顶层区域是最新的

4.2 有意义的工作和有意义的人际关系不仅是我们做出的美好选择，而且是我们天生的生理需求

人脑的构造先天地使人需要并享受社会合作

  • 很多数据显示人际关系是最好的回报，美好生活的基础是良好的人际关系

  • 人脑的结构

  • 个体利益与集体利益

  • 未来的人脑进化

4.3 理解大脑里的主要斗争，以及如何控制这些斗争，以实现『你』的愿望

只要频繁反复练习，你几乎可以养成任何习惯，产生自控力。好习惯让你实现『较高层次的自我』的愿望，而坏习惯是由『较低层次的自我』控制的，阻碍前者的实现。

我养成的最有价值的习惯是利用痛苦来激发高质量的思考。

  • 要明白，意识与潜意识在不断斗争

    • 灵感爆发于潜意识区域
    • 连接意识与潜意识

  • 要知道最长发生的斗争是情绪和思考的斗争

    • 杏仁核绑架

  • 调和你的情绪和思考

  • 善择你的习惯

    • 养成哪些习惯，戒掉哪些习惯
    • 习惯的养成：信号->行成常规->奖励

  • 坚持友善的训练『较低层次的你』，以养成好的习惯

  • 理解右脑思维和左脑思维的差别

  • 理解大脑可以改变的程度

    • 从身体锻炼到冥想
    • 不断练习，以及『习惯回路』
