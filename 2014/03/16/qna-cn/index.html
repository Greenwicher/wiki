Greenwicher's WikiQuestions and Answers（中文） 2014-03-16

Q Wixo 这名字有何含义？

A Wixo 意思是 A wiki theme for hexo 。

Q 如何指定某些页面生成 ToC(Table of Contents) ？

A 在文章头部添加 toc: true 。

Q 从哪里可以找到你的这几篇文档的 Markdown 源码？

A 在项目的 source 分支。

Q 为什么我有些笔记本打不开？

A 给你的笔记本/文章分类取名时不要加空格，例如 “饭 祷 爱” 。如果非加不可，可以考虑使用 _ ，例如 “饭_祷_爱” 。
